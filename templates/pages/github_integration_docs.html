GitHub Integration: scan every push automatically

Connect a repo once and Fora AI scans it on every push to the default branch — no manual uploads, no CI config files, no GitHub App to install.

What it does

Once a repo is connected, Fora AI watches for pushes to its default branch (usually main or master). Every time you push, we pull the latest snapshot of the repo, run it through the same scan engine used everywhere else in Fora AI, and save a full report — findings, severity breakdown, and a security score — to your dashboard.

Connect a repo in 3 steps

1

Create a GitHub Personal Access Token

Go to github.com/settings/tokens and generate a fine-grained token scoped to the repo you want to connect, with Contents: Read and write and Webhooks: Read and write permissions.

2

Paste it into GitHub Integrations

In your dashboard, open GitHub Integrations, enter the repo as owner/repo, paste the token, and click Connect repo. Fora AI verifies access and registers a push webhook on the repo for you.

3

Push code as usual

The next push to the default branch — from your terminal, an IDE, or even a direct commit in the GitHub UI — triggers a scan within seconds. No extra command, no CI file to add to your repo.

Where results show up

Every push scan appears in two places: the GitHub Integrations page, next to the repo it belongs to, and your normal Scan History — it counts toward the same plan limits and shows the same detailed report as any other scan.

Disconnecting a repo

Click Disconnect next to any connected repo. Fora AI removes the webhook from GitHub immediately and deletes the stored token — pushes to that repo stop triggering scans right away.

Security & privacy

  • Your token is used only to read the connected repo's contents and to create/remove its webhook — nothing else.
  • Every push notification from GitHub is verified with a per-repo signed secret before any scan runs, so nobody else can trigger a scan on your account.
  • Disconnecting a repo deletes the token and webhook from our side and from GitHub in the same action.
